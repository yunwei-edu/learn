第六篇horizon— Web管理界面
一、horizon 介绍：
理解 horizon
Horizon 为 Openstack 提供一个 WEB 前端的管理界面 (UI 服务 )通过 Horizone 所提供的 DashBoard 服务 , 管理员可以使用通过 WEB UI 对 Openstack 整体云环境进行管理 , 并可直观看到各种操作结果与运行状态。


DashBoard 与其他组件的关系


登陆 Dashboard 界面
启动客户端浏览器，于地址栏输入：
http://Horizone_server_ip/dashboard
